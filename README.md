Lab 10: Generic Media Library
Generic-контейнер для управления медиатекой (фильмы, эпизоды, подкасты).  

Особенности
Generics: Типобезопасное хранение любых объектов.  

Storage: Динамический массив с автоматическим расширением.  

Iterator: Собственная реализация Iterator<T> для обхода элементов.  

Iterable: Поддержка цикла for-each через интерфейс Iterable<T>.  

API: Методы add, get, remove, size и isEmpty.  

Testing: Покрытие базовой логики юнит-тестами JUnit 5.
